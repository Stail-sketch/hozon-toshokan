人事異動・退職・失踪 記録 / 澄奈市立保存図書館 / 昭和41年〜令和3年

本記録は、澄奈市立保存図書館の開館以来の職員の異動・退職・失踪に関する情報を時系列で整理したものである。白奈文香主任司書が閉館直前に人事台帳から転記した。

通常の退職・異動を除き、所在不明となった職員の一覧は以下の通り。
・昭和五十三年三月十四日：司書 河野忠男 / 出勤記録を最後に所在不明
・昭和六十一年三月十四日：司書 大沼裕子 / 夜勤明け以降の記録なし
・平成五年三月十四日：司書補 永井真一 / 地下書庫の整理中に所在不明
・平成十二年三月十四日：司書 小田切芳江 / 第三閲覧室の清掃担当。翌日不在
・平成二十二年三月十四日：事務員 宮下哲也 / 閉館業務中に所在不明

白奈メモ：日付を見てほしい。全員が三月十四日に消えている。十年ごと、正確に。昭和五十三年、六十一年は八年間隔だが、それ以降は規則的。そして令和三年三月十四日は、次の周期にあたる。

さらに確認すべき事実がある。三月十四日は奈瀬統一郎館長の誕生日であると同時に、先代館長（奈瀬家第十八代当主）の命日でもある。この日付が偶然に集中しているとは考えにくい。

松永司書の件を加えると、令和三年三月十四日に消えた職員は松永を含む八名に上る。過去の記録では一名ずつだった。なぜ今回だけ八名なのか。

白奈付記：私がこの記録を整理しているのは令和三年三月十三日。明日が三月十四日だということを、私は知っている。
