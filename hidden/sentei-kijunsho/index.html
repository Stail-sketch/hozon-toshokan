本文書は昭和38年に改訂された澄奈計画における処理対象者の選定基準を定めたものである。 昭和32年当初版より変更のあった項目には注記を付した。

基準1. 澄奈計画協定に関する情報を保持または漏洩しうる者
← 昭和32年当初より存在。適用件数 : 23件
基準2. 澄奈側（奈瀬家）の財産・権益に対して法的異議を申し立てた者
← 昭和32年当初より存在。適用件数 : 58件（最多）
基準3. 澄奈側（奈瀬家）と金銭トラブルを有し、公的機関へ申立を行った者
← 昭和35年追加。適用件数 : 41件
基準4. 儀式関連の目撃者または証言しうる者
← 昭和32年当初より存在。適用件数 : 19件
基準5. 澄奈側（奈瀬家）との個人的関係において信頼を失った者
← 昭和38年新設。適用件数 : 6件（本改訂時点）

上記各基準の「適用件数」を合計すると147件となり、処理台帳の総件数と一致する。

なお各基準の認定は全て澄奈側（奈瀬家）の申告に基づく。 彩霧班が独立して対象者を特定・認定した事例は本台帳において確認されない。 基準5の「信頼を失った者」については、具体的な判断根拠は奈瀬家の主観的申告のみである。
