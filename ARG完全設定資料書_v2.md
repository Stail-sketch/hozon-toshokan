## 👁️ 奈瀬家（なせけ）

### 表の顔
- 澄奈市の創設に貢献した名家
- 保存図書館の創設者・運営者
- 市民から尊敬される存在

### 裏の顔
- 明治時代から人間を過去に送る儀式技術を保有
- 技術を政府に提供→澄奈市の独立した支配権を獲得
- 政府要人を儀式で脅して裏から操作
- 政府を動かして邪魔な市民を消させていた
- 一族断絶により儀式の制御者がいなくなった

### 奈瀬家と国の関係
```
明治時代に儀式技術を政府に提供
↓
見返りに澄奈市の支配権を獲得
↓
政府要人を裏から操作
↓
邪魔な市民の消去を政府に実行させた
→ 表向きは「国の監視実験」に見える
```

## 🔮 儀式・技術の仕組み

```
大枠：儀式＋本系

特定の手順で本を「綴じる」儀式を行うと
その本が過去への扉になる

サブ要素：
・図書館の建物が儀式の増幅装置
　→ 外では効果が出ない
・奈瀬家の血を引く者が儀式を執り行う必要がある
　→ 一族断絶＝制御者の消滅
　→ 司書たちが意図せず巻き込まれた理由
```

## 📰 隠された歴史（ARGの核心）

### ストーリーの流れ
```
奈瀬家が澄奈市を実質的に作った
↓
市民に知られてはいけない秘密があった
↓
国を使って邪魔な市民を過去に送って消した
↓

↓

```

