# Claude Code用プロンプト ver2
## 澄奈市立保存図書館 ARG トップページ制作

---

## 最初に読んでほしいこと

このプロジェクトはARG（代替現実ゲーム）用のWebサイトです。
「実在する閉館した図書館のアーカイブサイト」に見せることが最重要です。
ゲームっぽくせず、リアルな公共施設サイトとして作ってください。
設定の詳細はARG完全設定資料書_v2.mdを参照してください。

## フェーズ1：トップページを作る

### デザイン仕様

**スタイル：90年代〜2000年代初頭の日本の公共施設Webサイト風**

#### 配色
```
背景色：#F5F0E8（くすんだ白・古い紙の色）
メイン文字色：#2C2C2C（濃いグレー）
アクセント：#1a3a5c（くすんだ紺）
サブアクセント：#8B4513（焦げ茶）
警告色：#8B0000（深紅）
区切り線：#B8A898
ヘッダー背景：#1a3a5c
ヘッダー文字：#F5F0E8
```

#### フォント
```
日本語：MS Mincho、游明朝、ヒラギノ明朝 ProN、serif
英数字：Courier New、monospace
```

#### レイアウト
- テーブルベースのレイアウト（昔のサイト感）
- 左サイドバーにナビゲーション
- 水平線（hr）多用
- 文字サイズ小さめ（12〜14px）
- 行間広め

## 必ず実装する仕掛け

### タイトルタグ
```html
<title>澄奈市立保存図書館 | あなたが読んでいる</title>
```

### HTMLコメント（ソースコードに埋め込む）
```html
<!-- 第7閲覧室の記録は削除済みです -->
<!-- 最終確認者：白奈文香 2021.03.13 23:47 -->
<!-- WARNING: このページのアクセスログは保存されています -->
<!-- このページは非公開のはずです -->
```

### JavaScript（main.js）
```javascript
console.log("なぜここを見ているのですか");
console.log("白奈文香は、まだ読んでいます");
console.log("ACCESS_LOG: 外部アクセスを検知しました");
```

### ページ上部に極小表示
```
このページは非公開のはずです
ACCESS_LOG : 外部アクセスを検知しました
```

### 訪問者カウンター風（フッター付近）
```
最終アクセス記録：████年██月██日
あなたは██人目の訪問者です
```

## 絶対に守ること

1. リアルに見えることが最優先（ゲームっぽくしない）
2. 説明しすぎない（不穏な要素は気づいた人だけ気づく程度に）
3. 黒塗り（████）は多用しすぎない（2〜3箇所のみ）
4. フィクション表記をフッターの目立たない場所に極小フォントで記載
5. 一見すると普通の閉館した図書館サイトに見えること
