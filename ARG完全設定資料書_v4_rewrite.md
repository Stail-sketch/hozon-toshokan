## 🗺️ 舞台：澄奈市（すみなしし）

- 架空の市
- 「すみなし」には「住みなし」「済みなし」「墨なし」の意味が重なる
- 奈瀬家が明治時代に実質的に整備した街
- 国の記録上では存在が曖昧で、資料によって表記が揺れる
- 市の中心部は整っているが、外縁に行くほど地図や記録が不安定になる

### 町の印象
- 表向きは静かな文化都市
- 古い行政文書や寄贈資料が異様に多い
- 住民は「奈瀬家」に敬意を持つが、詳しく語りたがらない
- 閉館後も図書館の噂だけがネット上で断続的に浮上する

## 👁️ 奈瀬家（なせけ）

### 表の顔
- 澄奈市の創設に貢献した名家
- 保存図書館の創設者・運営者
- 教育・文化事業の後援者
- 市民から尊敬される存在

### 裏の顔
- 明治時代から人間を過去へ送る儀式技術を保有
- 技術を政府に提供し、見返りとして澄奈市への強い支配権を得た
- 儀式を知った者、障害となる者を「記録から処理」する構造を作った
- 市民の失踪や記録改竄に国を関与させ、責任の所在を曖昧にしていた
- 一族断絶により儀式の制御者が消え、封印構造のみが残された

### 奈瀬家と国の関係
```
明治時代に儀式技術を政府へ提供
↓
見返りに澄奈市の支配権と機密保全を獲得
↓
国は「監視・処理」を実行する装置として関与
↓
表向きには「国の実験」だけが残る
↓
実際には奈瀬家が依頼側だった
```

## 🔮 儀式・技術の仕組み

```
特定の手順で本を「綴じる」儀式を行うと
その本自体が過去への扉になる

・図書館の建物は儀式の増幅装置
　→ 建物内部でのみ安定して作用する
・奈瀬家の血を引く者が執行者となる
　→ 一族断絶後は正常制御できない
・書庫・目録・蔵書検索が儀式の補助系統になっている
　→ 本を探す行為そのものが儀式に接続する
・記録を読むことは、過去の存在を一時的に現前させる行為になる
```

### 本作における重要な考え方
- **本は情報ではなく門である**
- **保存とは隠蔽でもある**
- **閲覧は調査であると同時に加担でもある**

## 📰 隠された歴史（ARGの核心）

### ストーリーの流れ
```
奈瀬家が澄奈市を実質的に作った
↓
市の成立そのものに隠された秘密があった
↓
秘密を知った市民は「記録から消去」された
↓
その痕跡は保存図書館の資料群に封じられた
↓
司書たちが異常を発見した
↓
司書たちも過去へ送られた、あるいは記録の側へ引き込まれた
↓
閉館後、蔵書は白紙化し、職員記録も崩壊した
↓
プレイヤーが読むことで、図書館は再び一時的に存在する
```

### 2段階どんでん返し構造

**第1段階（フェイクの真実）**
```
「国が澄奈市民を対象に監視・消去実験を行っていた」
「奈瀬家は監視対象か被害者だった」
```

**第2段階（本当の真実）**
```
「実は奈瀬家が国に実験・処理を依頼した側だった」
「消されたのは奈瀬家の秘密を知った市民や司書だった」
「奈瀬家こそが最大の黒幕だった」
```

### プレイヤー体験の軸
- 最初は「閉館した図書館の怪談」を追う
- 中盤で「国の実験施設」を疑う
- 終盤で「図書館そのものが隠蔽装置」だと気づく
- 最後に「自分が閲覧したことで図書館を再起動させていた」と知る

## 🌐 サイトの正体

```
本来は外部公開されるはずのない閉鎖系サイト
↓
奈瀬家断絶と封印崩壊により、断片的に外部から見えるようになった
あるいは白奈文香が過去から干渉し、意図的に浮上させた
↓
サイトそのものが「読むことで開く禁書」になっている
↓
プレイヤーがアクセスするたび、図書館の存在が一時的に補強される
↓
トップには「このページは非公開のはずです」と極小表示される
```

- 表向きは古い公共施設サイトの体裁を保つ
- ホラーサイトではなく、**普通の記録が少しずつ壊れている**見せ方を徹底する
- 隠しページだけでなく、表層にも軽微な異常を散らす

## 🗝️ ラビットホール設計（改訂版）

### 方針
- 難しすぎる一発勝負にしない
- 3手以内に一度は成功体験を与える
- 「調べれば進める」とプレイヤーに信じてもらう

### 基本フロー
```
白奈文香のX固定ポスト（突然復活）
↓
画像内に誰でも気づける違和感が1つある
　例：図書館名の一文字だけ旧字体、時刻の異常、蔵書ラベル番号など
↓
その違和感をきっかけにプロフィール・過去投稿を見る
↓
固定ポスト画像の軽い仕掛けから「#図書舘」を発見
↓
100投稿の中から#図書舘（舘）を3つ見つける
↓
3投稿の本タイトルまたは請求記号から短い暗号を解く
↓
URLの一部が判明
↓
固定ポスト画像の別要素から残りを補完
↓
hozon-toshokan.jpにアクセス
↓
トップページで「このページは非公開のはずです」
↓
ARG本編スタート
```

### 成功体験の置き方
- **1段目**：画像を見れば違和感に気づける
- **2段目**：3つの異常投稿を見つければ次に進める
- **3段目**：サイトに入った瞬間に「本当に辿り着いた」と分かる演出を置く

### 注意点
- ステガノグラフィは補助に留める
- 最初の導線を外部ツール依存にしすぎない
- ハッシュタグ誤字は一目で比較できる文脈を用意する
- URL完成後、トップページ内でもう1個小さな発見を返す

- 普通の本紹介：40
- 展示・館内案内：20
- 司書の日常・業務メモ：15
- 澄奈市や郷土資料への言及：10
- 奈瀬家資料への違和感：12
- 暗号投稿：3

### フェーズ設計
```
公開前
　Xアカウントを育成
　普通の投稿で人格を定着させる

公開直前
　投稿停止
　固定ポスト差し替え
　プロフィールやヘッダー画像に軽い異変を入れる

公開後
　固定ポストからラビットホール開始
　進行に応じて過去投稿が再解釈される
```

### 投稿の仕掛け
```
通常投稿：#図書館（館）
暗号投稿：#図書舘（舘）を3個だけ混ぜる
固定ポスト画像には、見つけやすい違和感＋補助的な隠し要素を仕込む
```

## ✅ この資料の目的

この資料は、世界観設定をまとめるだけではなく、**実際に公開してプレイヤーが離脱せずに遊べるARGにするための設計書**として使う。

そのため、今後の判断基準は以下の3点に置く。

1. 雰囲気はあるか
2. 調べれば進めるか
3. 発見するたびに物語が深まるか

この3つを満たす限り、設定の追加やページ数の増減は柔軟に調整してよい。
